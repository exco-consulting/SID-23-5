<<TMF_Application>>
Channel Guidance and Data Capture leads the specific channel front ending application with the captured information required. For each information element it can provide a list of valid options to select from (e.g. list of products available for the customer, list of options to select from a product). Finally it receives the captured information and publishes it.<br/>
