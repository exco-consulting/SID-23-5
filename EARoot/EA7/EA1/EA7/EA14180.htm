<<TMF_Application>>
Data Transformation and Formatting application works as a mediation layer to enable Data Management applications to provide seamless interface to make the data ready for storage and make it available for consumption as per application specific format. It also provides functionalities to secure data for e.g. (encryption of credit/debit card details) as defined by data compliance guidelines.<br/>
