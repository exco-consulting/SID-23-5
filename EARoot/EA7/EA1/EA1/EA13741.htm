<<TMF_Application>>
Contract Tracking and Storage provides the ability to view customer's existing contracts, search for customer contracts based on meta-data and to search text strings within contracts. There is a central repository for contract storage as well as the associated contract meta-data. This data can be mined for Campaigns and Lead Generation.<br/>
