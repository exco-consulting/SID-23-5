API Gateway Protocol Header Transformation function for awareness and support of Web protocols semantics (MIME types, HTTP Authorization and Caching headers, etc.) <br/><ul>
<li>Header transform with modifications of header names and/or header information (e.g., service enabler-specific header to IANA-based header) </li><li>Protocol transform: transformations of protocol to send API requests/responses (e.g., SOAP to REST) </li><li>Mash-up (Composition): assembling a new API with existing APIs provided by service enabler.</li></ul><p/>
