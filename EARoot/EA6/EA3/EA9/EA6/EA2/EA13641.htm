<<TMF_Function>>
API Gateway Event Routing function evaluates decision tasks making it possible to have decision trees within a workflow or policy evaluation, at any nesting level. This is the basis for many possible micro-workflow related activities, making possible to configure a decision point structure where multiple option are present e.g. Charging, Application Management (managing account information for recognizing applications and application developers).<br/>
