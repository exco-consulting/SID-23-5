Customer Digital Interaction Tracking Function tracks all user steps on the web site and influence on social networks: click-stream, viewed pages, consulted product descriptions, comments. In real time, this function analyses the user behavior, identifies his centers of interest and then deduces his profile which will be used to push contents and offers adequate for the user / customer needs.<br/>
