<<TMF_Function>>
Service Performance Data Analyzing function provides the necessary functionality to analyze the performance of the various service provider's services. This includes: <br/><ul>
<li>Analyzing performance data received from Service Performance Monitoring</li><li>Notify operations functions for automatic or manual action in case of alarming analysis results</li><li>Provide recommendations for performance improvements and trend analysis.</li></ul><p/>
