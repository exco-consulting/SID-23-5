<<TMF_Function>>
Service Test Reporting Rules Management Function includes:<br/><ul>
<li>Test rules defining the enrichment of test results. </li></ul>
  -   For example: enrichment of internet bandwidth measure based on installed product<br/><ul>
<li>Test rules defining the restitution of the result depending on test execution context</li></ul>
   -   For example: a test result of ok/not ok can be enough for a user whereas customer care requires more details in the result of the test.<br/>
