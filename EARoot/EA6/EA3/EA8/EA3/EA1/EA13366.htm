<<TMF_Function>>
Privacy Profile Type Creation provides a privacy dashboard function to define and create the individual privacy profile types by categorizing the Data Subject Parties,  defining the elements of the Privacy for the Privacy Profile Type, both initial and additional for future evolution.<br/>Note: Profiles Type are defined according to the Country Privacy Authority such as National Protective Security Authority (NPSA) for UK.<br/>
