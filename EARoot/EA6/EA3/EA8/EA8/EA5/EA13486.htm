<<TMF_Function>>
Permission Perimeter Configuration Function consists in specifying on which perimeter the permission carried by the Party Role applies on. A perimeter defines:<br/><ul>
<li>either an object or a set of objects such as a Product, a Billing Account, etc.</li><li>or using selection criteria to identify objects on which permission applies such as geographical area where the Distributor has right to sell.</li></ul><p/>
