<<TMF_Function>>
Resource Deployment and Retirement Function is responsible for implementing the resource strategy high level plan and develop the resource infrastructure (network, PFS, IT...). <br/><br/>The Resource Installed Base is updated and the resources are pre-configured and pre-activated if needed, for example for non-customer specific parameters, via the Resource Order Initialization and Installed Resource Management functions.<br/>
