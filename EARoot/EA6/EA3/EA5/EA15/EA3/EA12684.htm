<<TMF_Function>>
Resource Performance Data Analyzing functions provide the necessary functionality to analyze the performance of the various service provider's resources.  This includes: <br/><ul>
<li>Analyzing performance data received from Resource Performance Monitoring and historic performance data or system environment data </li><li>Notify operations applications for automatic or manual action in case of alarming analysis results</li><li>Determining the root causes of resource performance degradations.</li></ul><p/>
