<<TMF_Function>>
Resource Availability Validation function validates that the resource or resources specified on the resource order are available at the specified customer/service location and feasible from a network point of view. This includes the following:<br/>• Resource address validation <br/>• Resource availability validation <br/>• Resource feasibility validation <br/>• Establishment of service termination points <br/>• Determination of delivery interval<br/> It includes checking appropriate network facility route(s) according to engineering rules.<br/>
