Resource Task Item Sequence Control: Because of the “Resource Task Item Decomposition” of an orchestration request the result may be several actions that need to take place in a specific sequence. Each action is an item in the “Resource Task Item Sequence” and will perform according to a pre-configured configuration. The “Resource Task Item Sequence Control” controls the execution of each Item’s action.<br/>
