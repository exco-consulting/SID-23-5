<<TMF_AF>>
The Resource Order Orchestration Aggregate Function orchestrates the actions on the involved resources, managing dependencies between them.<br/>Example: for SIP VOIP delivery, depending of the chosen technical solution, the Resource Order Management component can orchestrate the update of the HSS, of the AS BASE and of the ENUM platform.<br/>It manages rollback, for example of the updates if one of the equipments can’t be updated, and prepares the notification with the global result of the Resource Order (OK or NOK).<br/>
