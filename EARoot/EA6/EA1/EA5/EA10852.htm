<<TMF_AF>>
Resource Diagnostic and Test Quality Loop Aggregate Function provides the necessary functionalities to manage a quality loop on diagnostic to continually tweak and improve it.<br/>Functionalities include:<br/>• Management of success/failure result feedback of diagnostic and correction provided by technicians or the systems after a correction<br/>• Offline analysis of the success/failure data compared to the diagnostic and correction rules to provide modifications to improve them. This analysis is done using statistical diagnostic data and test results to evaluate their relevance and the needed updates.<br/>The provided improvements can be used to update the definition of diagnostic rules and correction actions in the Resource Trouble Policy Management function and the definition of tests in the Resource Test Policy Management function.<br/>
