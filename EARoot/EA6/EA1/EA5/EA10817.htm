<<TMF_AF>>
Resource Test Management provides the necessary functionalities to manage the end-to-end execution of a test for a resource, either for manual or automatic testing in order to ensure that this resource is working properly. It includes:<br/>• Verifying test authorization and quota management<br/>• Carrying out test priority<br/>• Managing test scheduling<br/>• Managing bulk test<br/>• Retrieval of appropriate inventory data and setting up the test context and configuration<br/>• Test execution with access to the various resources required to perform resource testing<br/>• Managing resource test scenario<br/>• Test results restitution<br/>• Test results long term logging for offline data analyses. <br/>• Possible reporting of test results back to the initiator of the test. <br/>
