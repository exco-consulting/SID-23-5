<<TMF_AF>>
Resource Availability Management Aggregate Function ensures that a resource - or a set of resources – is available to deliver a service (CFS) order for example. It can check:<br/>• the state and current configuration of the resource(s)<br/>• its occupancy<br/>• its location<br/>• for an existing local loop if it is directly reusable from one end to the other, or partially un-built (and where it's necessary to rebuild)<br/><br/>This Aggregate Function can be used to confirm Service Availability (at least one set of needed resources is available) and/or the choice of the resources as part of the technical solution.<br/>
