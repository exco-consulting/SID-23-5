<<TMF_AF>>
The Resource Performance Supervision Aggregate Function enables resource functioning to be measured and supervised (in real-time or not), depending on rules defined by Resource Strategy Management, in order to prevent the resource performance degradation, with an on-the-fly analysis. <br/><br/>It includes: <br/>• filtering, aggregation<br/>• analysis of <br/>     • different types of information directly issued from equipment <br/>          • examples: usages, logs, flows, …<br/>     •  results of tests or historic of tests<br/>• it compares these information with expected values or pre-established limits<br/>• enrich or calculate resource performance indicators (KPI…) <br/>     • examples: Mean Time to Repair (MTTR), Mean Time Between Failure (MTBF)<br/>• analysis and evaluation of resource performance indicators evolution<br/>• it produces Resource Alarm <br/>     • when an information doesn’t correspond to the expected value<br/>     • threshold correlation: when a predefined threshold is exceeded, it is able to trigger a performance alarm which can be managed by Resource Alarm Management function.<br/>
