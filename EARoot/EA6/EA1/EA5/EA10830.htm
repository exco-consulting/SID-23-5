<<TMF_AF>>
The Location Management Aggregate Function permits to describe, identify and manage any type of location useful for the Service Provider operational processes.<br/>It includes:<br/>• geographical address description (city, street, number in the street, …), described using national or international codifications (for streets, countries, etc.),<br/>• buildings description (floors, rooms, racks, …), <br/>• geospatial information,<br/>•Information such as contact methods, opening time, access code, warnings, etc. can also be defined.<br/>
