<<TMF_AF>>
The Resource Supervision Policy Development Aggregate Function enables to define the policy concerning supervision (indicators and alarm) at Resource level.<br/><br/>It includes: <br/>• Definition of indicators (measures) associated with the resource type, with thresholds, calculation rules and consequences<br/>     • examples of indicators with threshold : pressure, temperature <br/>     • examples of indicators with calculation rules : storage occupation rate, load rate<br/>• Definition of rules <br/>     • for alarm activation<br/>          • for example if a dysfunction is detected in its power supply, or in the functioning of the elements which constitutes it (card, port, internal ventilator) and also if its temperature exceeds a threshold.<br/>     • for a Trouble ticket creation next to a resource alarm qualification.<br/>
