<<TMF_AF>>
The Resource Repository Reconciliation Aggregate Function defines, suggests to an administrator or directly applies rules to treat gaps between the Installed Resource Base and the current state or characteristics of Installed Resources identified by the Audit – Resource Discovery function. <br/><br/>These rules describe in which cases it is necessary to update the Installed Resource Base (ex: notification of a new equipment), via the Installed Resource Management function, or to update the equipment itself (ex: box)  - in some cases it can be necessary  <br/>• to trigger again the configuration of the installed resource <br/>• or even to trigger human intervention (ex: gap on physical network)<br/>
