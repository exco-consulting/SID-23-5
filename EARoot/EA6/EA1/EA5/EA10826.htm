<<TMF_AF>>
The Resource Usage Management aggregate function consist in follow-up of the Resource use, based on Resource. It deals with reports and analysis with an on-the-fly analysis (detailed & aggregated) of the resource uses and comparison with the resource capacity management and with the resource usage analysis forecasts.<br/>
