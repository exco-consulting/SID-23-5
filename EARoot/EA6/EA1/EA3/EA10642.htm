<<TMF_AF>>
Product Diagnostic and Test Quality Loop provides the necessary functionalities to manage a quality loop on diagnostic to continually tweak and improve it.<br/>This function provides an offline analysis of the usage of the tests to determine its relevance and provide modifications concerning the quota and authorized users. This analysis is done using statistical test usage data to evaluate the tests relevance and the needed updates.<br/>Following the analysis, the creation of additional product tests can be proposed or a test can be considered irrelevant and deleted.<br/>
