<<TMF_AF>>
Service Performance Management aggregate function groups functions that both proactively and reactively optimize the delivery of services to customers.  This can include a real-time, end-to-end view to ensure that each service is functioning correctly as well as a historical view. These functions build on the Resource Performance data and active end-end service performance test data to provide a view of a service.They provide a key input to determine the Quality of Service.<br/>
