<<TMF_AF>>
Service Diagnostic and Test Relevance Quality Loop Aggregate Function provides the necessary functionalities to manage a quality loop on diagnostic to continually tweak and improve it.<br/>Functionalities include:<br/><ul>
<li>Management of success/failure result feedback of diagnostic and correction provided by technicians or the systems after a correction</li><li>Offline analysis of the success/failure data compared to the diagnostic and correction rules to provide modifications to improve them. This analysis is done using statistical diagnostic data and test results to evaluate their relevance and the needed updates.</li></ul>
The provided improvements can be used to update the definition of diagnostic rules and correction actions in the Service Problem Policy Management function and the definition of tests in the Service Test Policy Management function.<br/>
