<<TMF_AF>>
The Service Order Delivery Preparation Aggregate Function allows in the context of a Service Order to:<br/>• query catalogues and installed bases to identify what needs to be delivered: resource specification and its configuration, service specification (CFSSpec), intervention (WorkSpec), supplier product, related to the ordered service,<br/>• prepare a Resource Order (CFS Order), a Service Order which will be delegated to another system, a Supplier Order, Stock Item Order or Work Order with the necessary information (the effective update in the order repositories will be supported by the corresponding Order Inventory Management functions)<br/><br/>In the case of a Service associated with an existing Internal Resource Type, this function also allows to: <br/>• check if a corresponding Installed Resource is operational in the resource installed base, and determine the operation to be performed at Resource level (creation or modification)<br/>• eventually group in the same Resource order several ordered services, based on the same CFS specification, and/or identified to be delivered at the same time by the Service Order Delivery Orchestration.<br/>
