<<TMF_AF>>
The Project Planning Aggregate Function handles all functions associated with the project management and not already implemented in other Aggregate Functions (resources management, budget management, …), that facilitate the work of the project managers.<br/>Sub-functions used:<br/>• Tasks and activities Building:<br/>Based on a multi-level Work Breakdown Structure (WBS), this function allows to define, organize and interface all activities of the project necessary to meet the predefined objectives of the project: delivery of the agreed scope (offer, product, …), quality, time and cost constraints.<br/>• Project Planning:<br/>Management of the project estimating, the means of the project optimizing, tasks schedule and identifying the critical path, evaluating and re affecting task over resources.<br/>
