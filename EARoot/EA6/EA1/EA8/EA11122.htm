The Analytics Information Support aggregate function groups functions involved in the collection, preparation and organization of information so that it can be used efficently and effectively in data analytics.  Their responsibilities include:<br/>1. Data Collection: Gathering, organizing, and storing relevant data from various sources.<br/>2. Data Quality Assurance: Ensuring the accuracy and reliability of collected data.<br/>3. Reference and Metadata Management: Defining reference data and dimensions that can be used to express the data consistently<br/>4. Data Transformation: Converting raw data into usable formats for analysis.<br/>5. Database Management: Maintaining data repositories and optimizing data access.<br/>6. Data Security: Implementing safeguards to protect sensitive information.<br/>7. Troubleshooting: Resolving data-related issues and providing technical support.<br/><br/>
