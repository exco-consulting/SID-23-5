Request Product Offering Purchase Return make a request to return one or more purchased product offerings. The request can be made in a number of ways, such as via the web, phone or email.<br/><br/>The request may be to send one or more offerings back for refund, exchange with another offering or offerings, or to replace one or more product offerings.<br/>
