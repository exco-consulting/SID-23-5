<<TMF_Process>>
The objective of the Close Resource Performance Degradation Report processes is to close a resource performance degradation report when the resource performance has been resolved.<br/>These processes monitor the status of all open resource performance degradation reports, and recognize that a resource performance degradation report is ready to be closed when the status is changed to cleared.<br/>
