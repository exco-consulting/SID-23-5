<<TMF_Process>>
This process triggers a well-defined action based on specific resource alarm event notification information as well as the non-arrival of resource alarm event notification information after a specific time interval has elapsed.  This process is also responsible for monitoring and triggering the appropriate action when a resource alarm event record is not cleared within a pre-defined period of time.<br/>
