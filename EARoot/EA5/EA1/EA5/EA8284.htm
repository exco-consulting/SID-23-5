Decommission Resouce Work Order Task business activity withdraws or removes a task safely and effectively from a resource work order. <br/><br/>Decommission Resource Work Order Task business activity ensures that resources work order tasks are properly taken offline or made inoperable/dismantles/withdrawn and removed in a way that avoids any risk. This includes preparing sub-tasks to remove the resource work order, preparing the resource to be removed, documenting the decommissioning, and inspecting resource has been safely and properly decommissioned.<br/>
