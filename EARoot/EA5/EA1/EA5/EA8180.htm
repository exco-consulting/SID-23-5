<<TMF_Process>>
The objective of "Validate Network Function Instantiation” process is to check a feasibility for deploying network function. This process grants an authorization of a lifecycle operation. When applicable, this process give a permission for reserving network function resources (e.g., virtualized infrastructure with location information) based on network function catalog.<br/>
