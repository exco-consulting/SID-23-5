Initiate Resource Work Order business activity starts a new work order for a specific resource along with all work orders tasks, roles and supporting resources that are need. <br/><br/>Initiate Resource Work Order establishes the need for a new resource work order and makes sure that work order is created sufficiently in a timely and accurate manner. It includes identifying the need for a resource, managing tasks necessary, creating the work order and communicating with relevant processes and parties.<br/>
