<<TMF_Process>>
This process is responsible for answering inquiries from other processes (like Customer-Ops), given a particular work task, about what timeframes the work can be performed in, considering dependencies between work tasks. This process does not look for a particular executor, but checks available work capacity in a particular time interval.<br/>
