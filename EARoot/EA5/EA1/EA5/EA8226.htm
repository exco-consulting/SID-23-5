<<TMF_Process>>
Manage Resource Order Work business activity directs and controls all work that are required to fulfill an approved resource order by ensuring the work related to the order is planned, executed and closed in a timely and efficient manner.<br/><br/>Manage Resource Order Work business activity includes activities to "Initiate Resource Work Order", "Create Resource Work Order", "Review Resource Work Order", "Plan Resource Work Order", "Close Resource Work Order", "Analyze Resource Work Order" and "Report Resource Work Order".<br/>
