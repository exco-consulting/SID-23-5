<<TMF_Process>>
Manage Data Integration View business activity direct and control tasks to provide a unified view of data by connecting many different sources of data, extracting that data, and presenting it in a suitable destination (such as a data lake or data warehouse).<br/> <br/>Manage Data Integration View involves identifying data integration entities, generating data integration profiles, relating data integration entities and designing data integration architecture. Altogether, this business activity improves the consistency of integrated data while providing a comprehensive view of data across the organization and its business operations.<br/>
