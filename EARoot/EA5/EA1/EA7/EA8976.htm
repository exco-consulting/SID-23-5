Metric Collection Job Specification specifies the time schedule for a data collection task ("job") for a Metric, applied to a selected set of objects. Metric Collection Job Specification defines the frequency, periodicity, or any time-based repetition of a measurement campaign. Metric Collection Job Specification also defines any additional constraints and conditions driving the Collection Job.<br/>
