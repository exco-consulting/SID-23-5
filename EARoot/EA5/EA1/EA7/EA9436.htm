Collect Reference Data business task is responsible for gathering reference data from various sources to be used in the organization.<br/> <br/>Collect Reference Data ensures that reference data is accurate, complete, and relevant to the organization and its operations. This involves defining the aim of collecting reference data, choosing the reference data collection method, planning reference data collection procedures, and collecting the reference data.<br/>
