<<TMF_Process>>
Specify Party Demographic Definition defines a type of demographic to be collected about parties and/or a role a Party plays. This includes specifying the permitted values associated with a demographic, such as age ranges, income ranges, as well as associations to  market segments and so forth. This process also enables changes to be made to any of these aspects for a demographic's definition.<br/>
