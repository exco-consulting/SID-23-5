Perform Assurance Analysis of Network state is responsible for an asynchronous process to review the state of a network and test that what has been configured is within the boundaries allowed by policies. In a persistent network this process can be executed occasionally as an audit, by a dynamic intermittent network will need constant validation and confirmation.<br/>
