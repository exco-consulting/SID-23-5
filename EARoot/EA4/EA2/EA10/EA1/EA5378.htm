Figure P.02 - Naming for Parties diagram
An individual is referred to by their name. This name can change over time (e.g. After marriage, by deed poll) and to allow for this we will model an individual’s name as a separate entity. Similarly, organizations can change their names (e.g. from Telecom to Telstra) and we will also model organization names as a separate entity. Note that this model does not provide support for individual name or organization name aliases. If aliases are required, they will be added to a later version of this model.<br/><i>“A name is an informal way of identifying an entity” [Fowler-AP]</i><br/>
