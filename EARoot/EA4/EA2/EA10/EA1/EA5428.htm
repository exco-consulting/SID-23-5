Figure P.10 - Party Profile Specification Entities diagram
A PartyProfileSpec defines the criterion used to group Parties that may typify MarketSegments or specifies the formulation of criteria for targeting of MarketingCampaigns. A PartyProfileSpec might be used to categorize PartyRoleSpecifications. In this case, the resulting PartyProfiles categorize PartyRoles else they categorize directly one or many PartiesTypes.<br/>A PartyProfileSpec can be defined for one or more GeographicAreas. <br/>The profile spec can target one or more ProductOfferings and one or more MarketSegments. <br/>PartyProfileSpecs can be based on (or defined by) demographic characteristics or any other type of characteristics (croterion).<br/>
