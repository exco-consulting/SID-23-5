Figure L.05 – I01 – Illustration of Physical & Logical Address diagram
<font color="#29313b">The term Address is often used to describe:</font><br/><ul>
<li><font color="#29313b"> Addresses that enable you to find a location (e.g. Street Address), which we will define as a physical address</font></li><li><font color="#29313b"> Addresses that don’t (e.g. PO Box address, email Address), which we will define as a logical address</font></li></ul>
<font color="#29313b">A logical address allows us to send something (tangible or intangible) to a central point where it can be retrieved. With email, a message is sent to the recipient’s ISP mail server. The recipient can then connect to the server from anywhere in the world and retrieve the message. The intermediate location is not of interest to the message sender.</font><br/><font color="#29313b">Only logical addresses that relate to Location (e.g. Po Box Address), will be covered in this model (see SID Party Domain model / Contact Medium for e-mail addresses etc.).</font><br/>
