A local coordinate system allows us to position things in a small area where the curvature of the Earth can be ignored. It would commonly be used to define placement in an Exchange building (Central Office)<br/><br/>It is an abstract entity that holds common attributes and relationships. <br/><br/>For continuative locations, the Units will usually be unity for a given measure (e.g. 1 metre). For grid coordinate systems, the unit will often be a non unity measure (e.g. 600mm)<br/>
