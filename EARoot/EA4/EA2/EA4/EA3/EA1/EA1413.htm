This class defines the semantics of the modifiesCFSSpec aggregation. Specifically, it enables an application to define which set of versions of this CustomerFacingServiceSpecification are appropriate for a given task.This aggregation represents the set of versions of this CustomerFacingServiceSpecification. The semantics of this aggregation are implemented with the CFSSpecVersionDetails class.<br/>
