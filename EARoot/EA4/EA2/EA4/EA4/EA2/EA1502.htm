<<TMF_BusinessEntity>>
This is an abstract base class that defines an integrated set of ResourceFacingServices that collectively meets the needs of a CustomerFacingService. For example, the Customer may have requested "GoldService", which is a ServicePackage that defines a set of ServiceBundles, each of which has its own QoS. A set of ResourceFacingServices can then be defined, one for each different ServiceBundle instance, that provides the required QoS for each ServiceBundle instance.<br/>
