A SIM Card is a tangible good, therefore this is a GoodsProductSpec.</font><br/><font color="#29313b">An evening & week-end product defines the usages’ tariff to be applied to the usage of the Mobile line, therefore this is a RatePlanProductSpec.</font><br/><font color="#29313b">A SMS package defines a predefined quantity of product usage; therefore, this is a FixedPricePackageProdSpec.</font><br/><font color="#29313b">A TV on Mobile ProductSpecification enables ProductUsageSpecs (like watch channel…), therefore this is a NetworkProductSpec.</font><br/>
