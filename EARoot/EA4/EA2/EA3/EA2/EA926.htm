Let’s consider a ShippingProductSpec that proposes the following choices:<br/><ul>
<li>Shipment Type: At Home, Pick-up & Go location, By-Hand by technician and By-Hand in Store</li><li>Delivery Time: Immediately, With Installation, 24h and 48h</li></ul>
In addition, a characteristic is needed to describe the shipping address.<br/>But all combinations of configuration are not available:<br/><ul>
<li>for At Home Shipment Type only Delivery Time 24h and 48h are available</li><li>for Pick-Up & Go location Shipment Type only Delivery Time 24h and 48h are available</li><li>for By-Hand by technician Shipment Type only Delivery Time With Installation is available</li></ul>
<ul>
<li>for By-Hand in Store Shipment Type only Delivery Time Immediately is available</li></ul><p/>
