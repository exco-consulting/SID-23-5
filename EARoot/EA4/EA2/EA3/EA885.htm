Figure [PrD-02] Product ABEs Level 1 & 2 diagram
The diagram below lists the Product Domain's ABEs Level 1 & 2.<br/><br/><b>Network Product ABE</b><br/><ul>
<li>The Network Product ABE contains the NetworkProduct description and relationships.</li></ul>
<b>Product Price ABE</b><br/><ul>
<li>The Product Price ABE describes all the prices applied to Products.</li></ul>
<b>Usage Volume Product ABE</b><br/><ul>
<li>The Usage Volume Product ABE contains the UsageVolumeProduct and all the entities specific to this type of product.</li></ul>
<b>Loyalty Program ABE</b><br/><ul>
<li>The Loyalty Program ABE contains all entities needed to instantiate a loyalty program for a specific customer such as LoyaltyProgram and LoyaltyAccount.</li></ul>
<b>Loyalty Program Specification ABE</b><br/><ul>
<li>The Loyalty Program Specification ABE contains all entities specifying a loyalty program such as LoyaltyProgramProdSpec and LoyaltyRule.</li></ul>
<b>Network Product Spec ABE</b><br/><ul>
<li> The Network Product Spec ABE contains the NetworkProductSpec description and relationships.</li></ul>
<b>Usage Volume Product Spec ABE</b><br/><ul>
<li>The Usage Volume Product Spec ABE contains the UsageVolumeProductSpec and all the entities specific to this type of product.</li></ul>
<b>Product Action ABE</b><br/><ul>
<li>The Product Action ABE contains Business Entities describing possible actions for ProductSpecifications, ProductOfferings and ProductConfigSpecs.</li></ul>
<b>Product Usage Spec ABE</b><br/><ul>
<li>The Product Usage Spec ABE specifies types of usage of ProductSpecifications.</li></ul>
<b>Product Catalog ABE</b><br/><ul>
<li>Product Catalog ABE is a collection of entities that represent the specification of ProductCatalogs and catalogs described by them that contain ProductOfferings and their associated ProductOfferingPrices.</li></ul>
<b>Product Offering Price ABE</b><br/><ul>
<li>The Product Offering Price ABE describes all the prices related to ProductOfferings.</li></ul>
<b>Product Offering Price Rule ABE</b><br/><ul>
<li>The Product Offering Price Rule ABE describes all rules related to prices concerning ProductOfferings.</li></ul>
<b>Pricing Logic Algorithm ABE</b><br/><ul>
<li>The Pricing Logic Algorithm ABE contains entities representing an instantiation of an interface specification to external rating function (without a modeled behavior in SID).</li></ul>
<b>Product Placement ABE</b><br/><ul>
<li>The Product Placement ABE gives a description of how ProductOfferings are marketed, such as via direct mail. A ProductPlacement contains qualitative statements, or assessments that describe such things as market penetration, product strategy, and so forth.</li></ul>
<b>Product Promotion ABE</b><br/><ul>
<li>The Product Promotion ABE describes the publication of a ProductOffering to encourage Parties to buy, rent, or in some other way procure the offering.</li></ul><p/>
