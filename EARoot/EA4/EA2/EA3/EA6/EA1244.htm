Figure Pr.34 - Usage diagram
The Usage class represents a usage event. It is described by the UsageSpecification and comprised of UsageCharacteristicValues that hold data attributes of a usage specified by UsageSpecCharacteristics. The UsageCharacteristic can either store its value in the usageCharValue property or reference predefined UsageSpecCharacteristicValue.<br/>
