Figure PrPo.01 – Illustration.03: K&H’s Heated Cat Bed diagram
<font color="#29313b">The second Figure depicts from which PartyRole Amazon orders the cat bed.</font><br/><font color="#29313b">The bed’s manufacturer is K&H as shown. The “K&H Heated Cat Bed” ProductOffering is a wholesale offer.</font><br/><font color="#29313b">In this illustration the same instance of ProductSpecification is used to describe the Kitty Cat Bed and used by both ProductOffering “Amazon Heated Cat Bed” and “K&H Heated Cat Bed”.</font><br/><font color="#29313b">But it could be useful to have two instances of ProductSpecification in order to be able to either extend its description or restrict the possible configurations of the purchased ProductSpecification (K&H may propose it in blue, red or orange, and Amazon may decide to propose only blue or red). In this case, the PartyRoleProdSpecRelationship would be used to qualify the relation between these two instances.</font><br/><font color="#29313b">There is no instance of a PartyRoleProductOffering entity that relates Amazon playing the reseller PartyRole to the K&H ProductOffering because Amazon does not offer the K&H bed on its web site.</font><br/><font color="#29313b">Amazon orders the cat bed that it lists for sale as shown in Figure 1 from K&H.</font><br/><font color="#29313b">K&H’s PartyRoleProductOffering orderTakenBy is set to true as the “K&H Kitty Cat Bed” ProductOffering is taken by K&H.</font><br/><font color="#29313b">The other attributes are also set to true to indicate that K&H has full involvement in fulfilling, revenue, and servicing orders that it receives from Amazon.</font><br/><font color="#29313b">This face is further re-enforced by the instance of PartyRoleProdOfferRelationship between the Amazon PartRoleProductOffering and the K&HpartRoleProductOffering for the cat beds. In this case the partyRoleProdOfferRelType is set to “ordered From” that indicates that Amazon orders the cat bed offering from K&H.</font><br/><i>Note: the arrows on the links express the orientation of the relationships.</i><br/>
