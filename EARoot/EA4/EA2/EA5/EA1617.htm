The diagram below lists the Resource Domain's ABEs Level 1 & 2.<br/><br/><b>Physical Resource Specification ABE</b><br/>        • The Physical Resource Specification ABE describes the invariant characteristics and behavior (attributes, methods, constraints, and relationships) of a PhysicalResource.<br/><b>Logical Resource Specification ABE</b><br/>        • The Logical Resource Specification ABE is used to define the invariant characteristics and behavior (attributes, methods, constraints, and relationships) of a LogicalResource.<br/><b>CompoundResource Specification ABE</b><br/>        • The CompoundResource Specification ABE used to define the invariant characteristics and behavior (attributes, methods, constraints, and relationships) of a CompoundResource.<br/><b>Resource Catalog ABE</b><br/>        • A collection of entities that represent the specification of ResourceCatalogs and catalogs described by them that contain ResourceSpecifications.<br/><b>Physical Resource ABE</b><br/>        • The Physical Resource ABE contains entities that are used to represent the various aspects of a Physical Resource.<br/><b>Logical Resource ABE</b><br/>        • The Logical Resource ABE contains entities that are used to represent the various aspects of a LogicalResource.<br/><b>Compound Resource ABE</b><br/>        • The Compound Resource ABE contains entities that are used to represent the various aspects of a CompoundResource.<br/>CompoundResource is a set of entities that shows how to aggregate PhysicalResources and LogicalResources into aggregate entities that has physical and/or logical characteristics and behavior.<br/><b>ResourceRole System View and TIP Resource ABEs </b><br/>        • not yet described<br/><b>Resource Number Portability ABE</b><br/>        • The Resource Number Portability ABE contains entities grouped for the portability itself. This means all the processes involved, like Port In, Port Out, Port Over, and Retirement of numbers.<br/><b>Resource Number Portability Role ABE</b><br/>        • The Resource Number Portability Role ABE contains entities related to the roles tasked for the portability.<br/><b>Alarm & Alarm Severity Assignment Profile ABEs not yet described</b><br/>
