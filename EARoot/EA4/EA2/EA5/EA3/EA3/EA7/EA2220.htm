<<TMF_BusinessEntity>>
This class defines the specification for the resource number to admit a completion of its characteristics, since the number is a very complex and broad entity, with specific things like format, ranges,area code, etc. This class can be thought of as a template, which represents a generic specification for implementing a particular resource number.<br/>
