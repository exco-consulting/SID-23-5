An InstalledSoftware (renaming the Software concept from previous releases) consists of encoded information or computer instructions deployed on a machine such that it can be executed. An InstalledSoftware is a SoftwareSpecification deployed using the SoftwareSupportPackage on platform which meets the HostingPlatformRequirements.<br/><br/>Each InstalledSoftware specifies their input and output respectively with SoftConnectionPoints.<br/><br/><br/><br/>Software may be nested within other software, thereby creating a containment relationship.<br/>
