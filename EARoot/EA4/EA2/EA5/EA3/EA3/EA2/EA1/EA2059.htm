This class defines the semantics of the PartyRoleUsesProcesses association. Since different PartyRoles have different privileges for working on and running the SoftwareResource, an association class is needed to accurately model these details. This association defines the semantics of how a particular PartyRole uses this SoftwareResource instance.<br/>
