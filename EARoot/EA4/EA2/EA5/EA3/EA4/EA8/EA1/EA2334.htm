This is a concrete class that represents any type of physical resource that can play a holding role. A common example is a Card that can also serve as a motherboard, or a Card that can serve as a daughter card that attaches to a motherboard. This class enables a single Entity, such as a Line Card, to have multiple roles - it can function as a Router Processor as well as a Holder.<br/>
