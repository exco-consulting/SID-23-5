Notes: This is a free-form string that defines the recommend power supply configuration required as a function of how populated the Equipment is. Often in networking devices, higher-capacity power supplies must be used in place of lower-capacity ones when the type and/or number of installed Cards exceeds a certain number. In addition, in order to have fully-redundant power, higher-capacity power supplies must sometimes be used, again depending on the type and number of Cards installed. This is an optional attribute.<br/>
