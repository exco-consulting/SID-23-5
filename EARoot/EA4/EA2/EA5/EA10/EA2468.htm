Figure SI.02-I01a - StockLocation in a Van diagram
<font color="#29313b">A service provider has technicians that will go to the customer’s house to install something.</font><br/><font color="#29313b">Let’s say it is a broadband product, or some entertainment product (Satellite TV / cable TV).</font><br/><font color="#29313b">In order to perform the install, the technicians have some equipment they must install (a set-top box, some cables, some plugs, maybe a specialized WIFI box, etc).</font><br/><font color="#29313b">In his van, he has some number of these items to be used for the installs for the day.</font><br/><font color="#29313b">These are stock items with stock levels, and located in a Van (StockLocation). Some countries call this “boot” stock, because it is in the boot or trunk of the vehicle.</font><br/><br/>
