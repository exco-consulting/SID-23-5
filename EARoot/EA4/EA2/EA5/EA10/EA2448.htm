A StockItem is further defined by characteristics and characteristics values.<br/>A StockItem might have several values for the same Characteristic, but only if there is a default Value (the value of the “pre-configured” StockItem corresponding to factory configuration). This configuration can only be modified for the ProductSpec by a ProductSpec applying on it.<br/>For example, the disk space of a TV decoder’s record capacity can be enhanced (realized through Software update), then we create a specific ProductSpecification for the Record Capacity applying on the TV decoder ProductSpec.<br/>
