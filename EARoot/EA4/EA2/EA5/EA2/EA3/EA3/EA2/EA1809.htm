HostingPlatformRequirementsSpec is used to represent the requirements that must be fulfilled by the platform on which the InstalledSoftware will run (e.g. compute, storage, network).<br/><br/>The derived association /SoftwareSpecificatonRequires is used to document the set of requirements of a platform which can host the concerned software.<br/>
