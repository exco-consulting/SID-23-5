<<TMF_BusinessEntity>>
A generalization that represents all attributes and associations that are common to Component and CompositeSoftwareSpecifications.  It also reprsents the definition of Software whose attribute values are shared by every instance of Software related to it.<br/><br/>A SoftwareSpecification specifies properties or associated information that characterizes a software, such as:<br/><br/>  - the release number, the status of the release, license conditions...<br/><br/>  - hosting platform requirements such as OS, disk and memory space...<br/><br/>  - input and output configuration and requirements<br/><br/>  - and includes all files required for deployment and installation such as installation manager<br/>
