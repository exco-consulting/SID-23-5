Figure SE.14 - Security Threat Indicator diagram
A Security Threat Indicator is activity that by itself isn’t harmful, but indicates malicious activity (e.g., a DNS call to resolve a known name known to be bad).  Threat indicators are activity a security analyst monitors for to detect attacks against its IT infrastructure.  <br/>
