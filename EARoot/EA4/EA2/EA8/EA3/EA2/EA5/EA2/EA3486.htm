<<TMF_BusinessEntity>>
The definition of a metric which may be categorized into three groups (Base, Temporal, and Environmental).  These three groups are further decomposed into groupings for exploitability and/or impact which define metrics used to score vulnerability severity.  <br/><br/><br/><br/>Notes: For example, Base Metrics are grouped into Base Exploitability and Base Impact Metrics.  Base Exploitability Metrics include Access Vector, Authentication, and Access Complexity Metrics.<br/>
