A standard value for a defined scoring metric (SecVulnerabilityScoringMetricDefn).  Each metric has multiple instances which represent multiple values.<br/><br/><br/><br/>Notes: For example,  Base Metric Access Vector has three pre-defined values - L/Local, A/Adjacent Network, and N/Network.<br/>
