It is the instance of a particular SecurityVulnerability that has been classified into one of the three categories (software flaw, software feature misuse, or software configuration issue). <br/><br/><br/><br/>Notes: For example (hypothetically), Internet Explorer could have a bug that is categorized as a software flaw.   There can be vulnerabilities that have one or more of the three category types associated with it.  In this case, the vulnerability would be scored for all three types.<br/>
