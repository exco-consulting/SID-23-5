A CriterionCalculationRule is a type of PolicyRule used to specify how to calculate the value of a Criterion.<br/><br/>The rule is evaluated when an event occurs. If the condition clause is satisfied, then the pass-action clause will be executed; otherwise, the fail-action clause will be executed.<br/><br/>In a CriterionCalculationRule, the action corresponds to the Criterion’s calculation and if needed the record of it.<br/>
