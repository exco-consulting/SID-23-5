<<TMF_BusinessEntity>>
This class is used to represent a single or set of Distinguished<br/>Name variable, which may include wildcards. This variable type is specifically defined for retrieving LDAP-based data. Thus, only DNValue classes can be used in the value portion of the condition expression with this PolicyVariable.<br/>
