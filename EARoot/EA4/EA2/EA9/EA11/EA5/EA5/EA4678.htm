<<TMF_BusinessEntity>>
This class is used to define a list of MAC addresses and MAC address range values as an unordered list of strings, each specifying a MAC address or a range of MAC addresses according to the canonical 802 MAC address format. Each string entry is either:<br/><br/> - a single MAC address, such as 0000:00A5:0000<br/> - a range of MAC addresses specified by an address<br/> and mask, such as 0000:00A5:0000,FFFF:FFFF:0000,<br/> which defines a range of MAC addresses in which the <br/> first four octets are equal to 0000:00A5.<br/>
