<<TMF_BusinessEntity>>
This is an abstract base class that represents how to form the action clause of a PolicyRule. This consists of a single occurrence of a PolicyStatement, which is of the form:<br/><br/> {variable, operator, value}<br/><br/>Policy actions have the semantics of "SET variable to value". There are two types of actions:<br/><br/> - pass actions are invoked if the condition clause was TRUE<br/> - fail actions are invoked if the condition clause was FALSE<br/>
