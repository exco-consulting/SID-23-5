The Policy Specification ABE contains entities that define the invariant characteristics and behavior of policy entities. This enables multiple instances to be derived from a single specification entity. In this derivation, each instance will use the invariant characteristics and behavior defined in its associated template.<br/><br/>This domain includes specifications for a PolicyRule, its components, and how multiple PolicyRules can be grouped together.<br/>
