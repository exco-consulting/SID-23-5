Figure An.06 - AnomalyGrouping Characteristics & Entity Associations diagram
This diagram shows relationship between AnomalyGrouping , AnomalyGroupingSpecification and associated Characteristics.<br/>AnomalyGroupingSpecCharacteristic (derived from CharacteristicSpecification in Characteristic ABE) is used for defining a particular characteristic and the potential values of this characteristic are defined using the entity AnomalyGroupingSpecCharacteristicValue (derived from CharacteristicSpecValue in Characteristic ABE) . The possible values defined using AnomalyGroupingSpecCharacteristicValue is associated with AnomalyGroupingSpecCharacteristic using relationship AnomalyGroupingSpecCharacteristicEnumeratedBy.<br/>The use of AnomalyGroupingSpecCharacteristic within the AnomalyGroupingSpec is attributed through AnomalyGroupingSpecCharUse.<br/>AnomalyGroupingSpecification and AnomalyGroupingSpec are associated using the relationship AnomalyGroupingSpecFurtherDefinedBy<br/>Similarly AnomalyGroupingSpecCharUse is associated with a particular AnomalyGroupingSpecCharacteristic using relationship AnomalyGroupingSpecCharacteristicFurtherDefinedBy<br/>AnomalyGroupingSpecCharValueUse is used to define the particular variation of the AnomalyGroupingCharUse with the specific value defined using the AnomalyGroupingSpecCharacteristicValue<br/>The relationship between the AnomalyGroupingSpecCharacteristicValue and AnomalyGroupingSpecCharValueUse is defined using the association AnomalyGrupingSpecCharValueFurtherDefinedBy<br/>AnomalyGroupingSpecCharValueUse is associated with AnomalyGroupingSpecCharUse using relationship AnomalyGroupingSpecUseEnumeratedBy relationship<br/>AnomalyGroupingSpecification uses the PolicyRuleSpecification to evaluate constraints that is applicable while defining the grouping logic<br/>AnomalyGrouping associated with Anomaly is described by AnomalyGroupingCharacteristicValue (derived from CharacteristicValue) using the AnomalyGroupingDescribedBy relationship, and this entity is further related to AnomalyGroupingSpecCharacterisitc and AnomalyGroupingSpecCharacteristicValue through associations AnomalyGroupingSpecCharacteristicDescribes and AnomalyGroupingSpecCharacteristicValueInstantiatedAs<br/>
