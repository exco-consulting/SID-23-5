Figure PF.05 - Performance Interval diagram
PerformanceIntervals also apply to Performance entities as shown in Figure PF.05.  It would be incorrect to assume that, since the PerformanceSpecification entity is associated to the Performance entity and PerformanceSpecification is associated to PerformanceSpecificationInterval, the interval association to Performance could be derived using this association.  This assumption is not correct because a PerformanceSpecification can be associated to one or more PerofrmanceSpecifictionIntervals.  From a Performance perspective, the PerformanceSpecificationInterval defines the time period over which the Performance is computed.<br/>
