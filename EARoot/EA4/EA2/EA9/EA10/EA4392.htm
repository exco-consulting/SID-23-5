Figure PF.17 - Performance Adhoc Collection diagram
Ad-hoc collection is an on-demand collection method of Performance measurements results from a consuming application, requesting spontaneously a set of data for a set of network or service testing resources for a given temporal context. As the ad-hoc query is a single isolated request, its lifecycle is much simpler and isolated, having a request-response nature.<br/>The parameters of the PM data request are very similar to the parameters of the measurement job; the main difference is related to the administration of the request.
