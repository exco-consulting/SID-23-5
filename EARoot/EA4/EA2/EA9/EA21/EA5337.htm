Directed edge specify relationships between Vertices in a graph . They usually have a type -such as port contained within a slide-in-unit, powered by , etc. Practical use of Directed Edge in graph are to provide restricted (subset ) views of all the multiple relationships between vertices. . In the TMF920A Topology Discovery Service API Component Suite  and the TMF686 Topology API these edges can be derived for information in multiple dissimilar inventories but normalized to a standard set of types to support many common processes such as anomaly detection , establishing availability of equipment at specific locations to support provisioning<br/>
