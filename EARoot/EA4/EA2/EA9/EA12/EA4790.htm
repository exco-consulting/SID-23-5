<<TMF_BusinessEntity>>
The ProjectScope defines the "functional area" impacted by a project that can be expressed by eTOM process elements, SID entities and/or TAM application which might be effected by the project. In a multi-project environment (or a program) the scopes of those projects might be structured in a (composite) hierarchy.<br/>
