Figure CP.01 – I01 - Resource and Configuration Examples diagram
<font color="#29313b">The following Figure provides a simple example of the concepts just introduced.</font><br/><font color="#29313b">A Connection Termination Point (CTP) is constrained by the Resource Specification at the top-left. The CTP points to additional properties (attributes) in Resource Configuration (perhaps properties specific to some transport layer such as Ethernet). The Resource Configuration Specification (top right) constrains the Resource Configuration. In the example, there is some change in the Resource Configuration Specification which in turn changes the Resource Configuration instances (the version number is changed to 2).</font><br/><font color="#29313b">In addition to being applied to the various version of Resource Configuration, Resource Configuration Specification can also be applied to CTP.</font><br/><font color="#29313b">It is possible alternate between different versions of Resource Configuration (e.g., based on time of day).</font><br/>
