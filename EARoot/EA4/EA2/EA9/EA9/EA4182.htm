Figure Me.07-08-I01 – Simple consequence diagram
<font color="#29313b">The following examples illustrates the use of MetricDefMeasureThresholdSet, MetricDefMeasureThresholdSet and MetricDefMeasureConsequence.</font><br/><font color="#29313b">A Metric is defined to evaluate the “Average Layer 3 packet roundtrip delay in millisecond”</font><br/><ul>
<li><font color="#29313b">5 value ranges (thresholds) in millisecond are defined for that Metric</font></li></ul>
<ul>
<li><font color="#29313b">[0;100[ Good</font></li><li><font color="#29313b">[100;250[ Acceptable</font></li><li><font color="#29313b">[250;1000[ Bad</font></li><li><font color="#29313b">[1000;+¥[ Very Bad</font></li><li><font color="#29313b">nil value Not measurable </font><br/><font color="#29313b">(may be due to problems such as disconnect or due to sensor failure)</font></li></ul>
<ul>
<li><font color="#29313b">for each value range one or more Consequences to trigger might be defined:</font></li></ul>
<ul>
<li><font color="#29313b">[0;100[ Green GUI</font></li><li><font color="#29313b">[100;250[ Green GUI</font></li><li><font color="#29313b">[250;1000[ Light Orange GUI & raise a Warning notification</font></li><li><font color="#29313b">[1000;+¥[ Dark Orange GUI & raise a Major notification</font></li><li><font color="#29313b">nil value Red GUI & trigger the diagnose of “not measurable”</font></li></ul>
<ul>
<li><font color="#29313b">• Some consequences are defined according to the previous measure’s value range</font></li></ul>
<ul>
<li><font color="#29313b">no longer [0;100[  and whatever the new value is notify degradation</font></li><li><font color="#29313b">no longer [1000;+¥[  and new value [0;250[ notify end of Major trouble</font></li></ul>
<font color="#29313b">Rk: To have more simple diagrams the entities used to detail the relationships are not represented (MetricDefMeasureThreshConsDependency and MetricDefMeasureThresholdConsequence) in the illustrations.</font><br/><font color="#29313b">Here for any measure in the range of [0;100[ the consequence “update GUI to green” is triggered.</font><br/>
