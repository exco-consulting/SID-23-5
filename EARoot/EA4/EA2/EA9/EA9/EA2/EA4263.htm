Figure A-5 Inheritance structure of metric objects diagram
<font color="#29313b">Several subclasses of scanner, each deriving gauge values using various statistic algorithms, are specified.</font><br/><font color="#29313b">The Scanner class has an attribute for the granularity period</font><br/><font color="#29313b">The Monitor metric class adds attributes for the observed Managed Object pointer, the name of the observed attribute, and the derived value. </font><br/><font color="#29313b">The Mean Monitor class adds attributes for the Moving Time Period (MTP) over which values are scanned, and the derived estimate of the mean taken over the MTP.  The algorithm used to specify the mean is not specified.</font><br/><font color="#29313b">The Algorithm Indicating Mean Monitor class adds an attribute which specifies the algorithm used to calculate the estimate of the mean. </font><br/><font color="#29313b">The Moving average mean monitor class explicitly specifies using an Exponentially Weighted moving average (EWMA) algorithm to calculate the estimate of the means.  </font><br/><font color="#29313b">The three subclasses of Moving Average Mean Monitor class add EWMA estimates for percentile, variance, and min/max statistics.</font><br/>
