<font color="#29313b">A DimensionalMeasure produces a single numerical value, expressed using a unit of measure.  SMM specifies three subclasses of Dimensional Measure:</font><br/><ul>
<li><font color="#29313b">A Direct Measure is one which does not involve a composition of subsidiary measures.  It applies an operation to the measured entity.</font></li><li><font color="#29313b">A Binary Measure is one which evaluates a binary expression (i.e., sum, difference, ratio) involving two subsidiary Measures.  </font></li><li><font color="#29313b">A Collective Measure represents measures which when applied to a given entity accumulates measurements of entities similarly related to the given entity.  For example, it can evaluate an accumulator function across a set of subsidiary Measures taken on measurands which are components of a parent measurand.  The accumulator functions specified are sum, maximum, minimum, average, and standard deviation. </font></li></ul>
<font color="#29313b">The subclasses of MeasureRelationship express the relationships necessary to evaluate Binary and Collective Measures.</font><br/><font color="#29313b">The measurandQuery association end of MeasureRelationship specifies a query that is used to determine the measurands that satisfy the relation between two measures.</font><br/><font color="#29313b">SMM also defines a parallel structure of Measurement relationships, as shown in Figure A-3.</font><br/>
