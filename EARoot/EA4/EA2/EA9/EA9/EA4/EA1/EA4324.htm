A MetricDefMeasureThresholdSet provides a way to group a set of one or more MetricDefMeasureThresholdRules specifying the condition to evaluate for applying consequences.<br/><br/>A MetricDefMeasureThresholdSet is used to evaluate one or more MetricDefMesures.<br/><br/>A MetricDefMeasureThresholdSet contains either MetricThresholdRuleDefinition instances or PreDefinedMetricThresholdRule instances.<br/>
