<font color="#29313b">An EntityIdentificationSpecification represents one or more attributes (not attribute values) used to recognize (identify) an instance of an Entity.  Figure EI.04 – EntityIdentificationSpecification Examples below shows an 8-byte Network Site Format using ANS T1.253. </font><br/><font color="#29313b"><i>Note that the various component specifications (Geographical Code, Geopolitical Code, and Network Site Code) that make up a complete Network Node ID, which is represented as an CompositeEntityIdentificationSpec object, are represented as instances of the ComponentEntityIdentificationSpec, and are aggregated by the CompositeEntityIdentificationSpec.</i></font><br/>
