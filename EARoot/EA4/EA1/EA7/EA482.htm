<<TMF_ABE>>
The Resource Facing Service Spec Role ABE defines a ServiceSpecification, in terms of a set of ServiceSpecificationRoles, for a ResourceFacingService. It is used to represent the shared characteristics of a ResourceFacingService. This enables the ResourceFacingService to be managed abstractly using ServiceSpecificationRoles. It also helps define a ResourceFacingServiceSpecification in terms of the functions that it has or provides.<br/>
