<<TMF_ABE>>
The Resource Topology ABE contains entities that define physical, logical, and network topological information. This information is critical for assessing the current state of the network, as well as providing information on how to fix problems, tune performance, and in general work with the network (both as a whole and with its components). Each of these topological views provides its own physical, logical, or network related information that can be used to manage one or more layers in a layered network.<br/>
