<<TMF_ABE>>
The Resource Specification ABE contains entities that define the shared characteristics and behavior of each of the four types of Resource entities. This enables multiple instances to be derived from a single specification entity. In this derivation, each instance will use the shared characteristics and behavior defined in its associated template.<br/>
